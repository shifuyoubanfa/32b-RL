# judgecal 结果 · Kimi 对"换词复述照抄"的分辨率

**一句话**：Kimi 平均上能按"照抄多少"排序（Spearman 0.95），但**只在"重度照抄"那一端有真分辨力**；干净到中等（0~3 句照抄）这一段它打分又重又飘、相邻档分不开，而且**把干净的也压低分**。所以强化学习只能用"干净 vs 明显脏"的大间距对子，细分指望不上。

- 灰虚线 = 理想（Kimi 该跟着真实档位走）；蓝线 = Kimi 实际打分（均值 ± 标准差）。
- 红色阴影 = 曲线压平、相邻档分不开的区段。

## 怎么读（4 点）

1. **整体单调、但系统性偏重**：照抄越多分越低（对），可连"没抄"也只给到 **7.3**（不是 10），整条线压在理想线下面——Kimi 偏严。
2. **顶端飘得离谱**："没抄"的标准差 **1.93**，有的全干净 think 被打到 **2.75**（当成挺脏的）。这不是打分噪声（那只 0.55、打几遍能压掉），是**不同样本之间 Kimi 自己就不一致**——多打几遍也救不了。
3. **底端一刀切到 0**：抄 4 句(0.03)和完全照抄(0.0)都被砸到 ~0，分不开。
4. **只有一个干净台阶**：抄3句(1.84) → 抄4句(0.03) 这一跌是唯一"分得开"的相邻档。其余相邻档（没抄~抄3、抄4~完全）全压平。

## 对强化学习的意思（= 天花板）

- **可用**：把"基本干净"和"重度照抄(≥4 句/完全)"配成大间距对子去训——这段 Kimi 分得开。
- **不可用**：0~2 句这种细微照抄差，Kimi 分不出、还乱压分；**没抄 ↔ 抄1句 分不开**，意味着它会把"只是用了事实"的干净 think 误当照抄（阶段7 老病在打分层面复发）。
- **多打几遍 K 解决不了**：顶端的飘是样本间分歧（k 压不掉）。真要细分，得换"同题两两比较"，或者就接受只做大间距对子。

## 旁证（分工对得上）
